microChess — Exploração Visual v1

Três direções de arte aplicadas a quatro telas-chave (Menu, Matchmaking, Jogo, Duelo). Comparativo lado a lado para escolher a direção vencedora antes de detalhar o sistema completo.

Referência: layouts atuais Viewport 9:16 · mobile-first

Ponto de partida. Hoje o jogo usa preto #080808 + dourado #d4a832 com Cinzel/Cinzel Decorative/IBM Plex Mono. A identidade está ali, mas diluída em telas heterogêneas (botões grandes demais, hierarquias chapas, espaços ociosos).

Objetivo desta exploração. Testar três DNAs completos — não só paletas diferentes, mas linguagens diferentes: como botões se comportam, como a tipografia hierarquiza, como o tabuleiro se apresenta, como o duelo comunica drama.

Como ler. Cada coluna é uma direção. As linhas comparam a mesma tela. Use os Tweaks (canto inferior direito) para focar em uma única direção em tamanho maior.

Direção A · Futurista

Arena Neon

Interface de jogo competitivo moderno (Valorant, League). Grid técnico, cyan glow, tipografia geométrica, HUD como painel de controle.

Direção B · Medieval / Clássica

Códice Dourado

Amplifica o DNA atual até o fim: pergaminho, ornamentos, selos, letras capitulares. O tabuleiro vira uma mesa de jogo antiga iluminada por velas.

Direção C · Moderna / Minimalista

Editorial Claro

Fundo creme, tipografia editorial, xadrez como produto digital limpo (Linear, Things, Duolingo). Um acento laranja para ação. Sem drama visual.

AArena Neon · rationale

Quando faz sentido. Se você quer posicionar o microChess como jogo competitivo (ranked, MMR, stats) — esta linguagem comunica isso em 2 segundos.

  • Tipo: Space Grotesk + JetBrains Mono (técnico, legível em telas pequenas)
  • Cor: cyan #5EF2FF como acento único; violeta #A078FF só no oponente
  • Motifs: grid técnico, cantos cortados, números indexados (01, 02), telemetria em mono
  • Risco: genérico no gênero "esports" se não for bem dosado

BCódice Dourado · rationale

Quando faz sentido. Dobra a aposta no DNA atual. Posiciona o jogo como uma relíquia digital — um tom artesanal que destaca dos xadrezes genéricos da loja.

  • Tipo: Cinzel Decorative (display) + Cormorant Garamond (corpo, mais legível que Cinzel) + IBM Plex Mono (UI)
  • Cor: Dourado #D4A84A com profundidade (gradientes bronze), acentos carmim #8A2A1A para combate
  • Motifs: selos, ornamentos (✦), cantos de moldura, dados físicos, shield/crest no matchmaking
  • Risco: pode ficar carregado se aplicado em excesso; escalar ornamentos com cuidado

CEditorial Claro · rationale

Quando faz sentido. Se a meta é Play Store e retenção de casuais. Linguagem de app produto, familiar a qualquer usuário de iPhone/Android moderno.

  • Tipo: Inter (corpo) + JetBrains Mono (números) — stack neutro e moderno
  • Cor: Fundo creme #F5F3EE, tinta #18130C, acento laranja #FF5A1F para ação
  • Motifs: cartões arredondados, avatar inicial, separadores finos, peças em Unicode com contorno (não dependem de SVG)
  • Risco: joga fora o charme atual; precisa ser compensado com micro-interações fortes
